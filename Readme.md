# cDock Themes

Some themes for [cDock 2](http://w0lfschild.github.io/app_cDock.html) (a dock customization app for OS X 10.9 and above).

## Installation

Move the theme folders to `~/Library/Application Support/cDock/themes/` and restart cDock.
